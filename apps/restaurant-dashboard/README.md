# @hashtap/restaurant-dashboard

Restoran yöneticisinin laptop/tabletten kullandığı panel.

## Sayfalar

- **Siparişler** — canlı sipariş akışı (WebSocket), durum güncelleme.
- **Menü** — POS'tan gelen iş alanları (salt-okunur) + HashTap alanları (düzenlenebilir).
- **Analiz** — ciro, yoğunluk, ürün satış, masa dönüşü (recharts).
- **Ayarlar** — POS bağlantısı, iyzico subMerchant, e-Arşiv, konsept/masa.

## Durum

İskele. API uçları tamamlanınca sayfalar doldurulacak.
